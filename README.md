# tts-telegram-bot
Telegram bot that convert text to voice message.

# Run
```zsh
git clone https://github.com/zetsub0/tts-telegram-bot.git
cd tts-telegram-bot
go mod tidy
make run
```

# Supported languages
- Arabic
- English
- Russian

# Limits
Something like 120 symbols.\
q: Why so bad?\
a: There are no good free tts APIs.
